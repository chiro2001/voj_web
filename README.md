## How to start

Make sure you have installed the latest version of [Node.js](https://nodejs.org/), `yarn` and `npx`. If `npx` and `yarn` are not installed, run `npm install -g npx`, `npm install -g yarn` to install them.

### Install dependencies

```shell
cd crosst
yarn
```

### Run

```shell
yarn start
```

### Build static files

```shell
yarn build
```
